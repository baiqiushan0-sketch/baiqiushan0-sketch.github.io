# 手势测试 Skill
用预录的手部骨骼数据测试手势识别逻辑，无需真人摄像头。

## 使用方式
当用户说 "测试手势"、"验证手势识别"、"检查比1/比2是否正确触发" 时触发。

## 手势测试数据
存储在 homepage/test-landmarks/ 目录中，每种手势一个 JSON 文件：
- fist.json（握拳）
- index.json（比1，仅食指伸展）
- peace.json（比2，食指+中指伸展）
- three.json（比3，食指+中指+无名指伸展）
- open.json（五指张开）

每个文件包含一组 MediaPipe 格式的 21 点 landmark 数据。

## 步骤
1. 读取对应的 JSON 测试数据
2. 注入到页面中触发手势识别
3. 验证是否正确切换到目标粒子模式
4. 截图对比验证
