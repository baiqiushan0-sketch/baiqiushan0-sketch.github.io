# 数据可视化 Skill
将数据快速转化为交互式图表。

## 使用方式
当用户说 "画个图"、"可视化"、"图表"、"折线图"、"柱状图"、"饼图"、"仪表盘" 时触发。

## 工具
- ECharts（交互式图表，优先）
- Plotly（备选，3D 支持好）
- Python matplotlib（静态图表）
- 输出 HTML 文件，浏览器直接打开

## 能力
1. 折线图、柱状图、饼图、散点图、雷达图
2. 热力图、地图、桑基图、关系图
3. 实时数据更新仪表盘
4. 多图联动/对比
5. 导出 PNG/SVG
6. 深色主题适配

## 步骤
1. 理解数据结构
2. 选择最合适的图表类型
3. 生成独立 HTML 文件（内嵌 ECharts CDN）
4. 浏览器打开预览
5. 支持导出图片

## 常用场景
- 时间序列 → 折线图
- 分类对比 → 柱状图
- 占比 → 饼图/环形图
- 分布 → 散点图/直方图
- 关系 → 力导向图/桑基图
